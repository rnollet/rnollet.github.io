Sandbox and personnal website
=============================


Comment faire un beau site web ?
--------------------------------

Le challenge étant de combiner :
- **Ne pas y passer trop de temps** Parce que ça n’est pas essentiel. Le but
  est que les gens y trouvent les infos qu’ils cherchent, pas qu’ils passent
  des heures à s’extasier devant.
- **Que ce soit beau** Que ça fasse professionnel. Que ça n’ait pas l’air d’un
  vieux markdown tout moche.
- **Que ce soit simple** Que je ne me perde pas dans du code auquel je ne
  comprends plus rien ou dans une API obscure.
- **Que ce soit portable** Que si je change d’hébergeur, ou si je veux le
  mettre sur la page du site de mon institution, etc. ce soit juste faisable.
